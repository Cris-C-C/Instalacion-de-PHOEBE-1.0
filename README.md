# PHOEBE 1.0 Installation
PHysics Of Eclipsing BinariEs (PHOEBE) es un software diseñado para mejorar y extender el modelo de Wilson–Devinney, y es ampliamente utilizado en astronomía para el análisis detallado de la estructura, las propiedades y la evolución de sistemas binarios eclipsantes. Aunque PHOEBE ha sido actualizado a lo largo de los años hasta la versión 2.4, su uso puede resultar complejo para usuarios que se inician, ya que se encuentra implementado completamente en Python. Por esta razón, la versión 1.0 de PHOEBE resulta más accesible, pues ofrece la comodidad de una interfaz gráfica, permitiendo un enfoque más práctico en el análisis de estos sistemas; sin embargo, debido a su descontinuación, asociada a la antigüedad de esta versión, se presenta este tutorial con el objetivo de facilitar su instalación y uso.
